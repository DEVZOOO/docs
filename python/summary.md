## 사칙연산 : 더하기 `+`, 빼기 `-`, 곱하기 `*`, 나누기 `/`, 몫 `//`


## 반복문
같은 패턴의 명령을 반복해서 실행한다.

### 반복문 for
```python
for 변수 in 반복범위 :
    실행문1
    실행문2
    ...
```
- 반복범위 : 숫자, 문자열, 리스트, ... 가능하다!
- range(start = 0, end, step = 1)   
반복 범위 지정  
start, step 생략 가능  
    |값|기본값|
    |:---:|:---:|
    |start|0|
    |step|1|
    ```python
    range(5)        # 0, 1, 2, 3, 4
    range(1, 5, 2)  # 1, 3 (1부터 5 전까지 2만큼 증가)
    ```
- 예시
    ```python
    # ex1. 숫자
    for i in range(3) :
        print('%d번째 루프' %i)
    # 0번째 루프
    # 1번째 루프
    # 2번째 루프

    # ex2. 문자열
    for ch in 'abcd' :
        print(ch)
    # a
    # b
    # c
    # d

    # ex3. 리스트
    for item in ['월', '화', '수', '목'] :
        print(item)
    # 월
    # 화
    # 수
    # 목
    ```
