## 사칙연산 : 더하기 `+`, 빼기 `-`, 곱하기 `*`, 나누기 `/`, 몫 `//`
계산기에 입력할 때 처럼 숫자와 기호를 입력하면 결과값이 출력된다.   
```python
>>> 3 + 5
>>> 10 - 7
>>> 8 * 6
>>> 10 / 3 # 3.3333...
>>> 10 // 3 # 3
```
